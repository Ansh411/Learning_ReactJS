# CSS Libraries

### SCSS/SASS

A powerful CSS preprocessor that adds variables, nesting, mixins, and functions.<br/>

Helps write cleaner and more maintainable CSS.<br/>

Compiles down to normal CSS before running in the browser.
<hr/>

### Bootstrap

A popular CSS framework with ready-made components like navbars, grids, buttons, and modals.<br/>

Uses a 12-column grid system for layout.<br/>

Great for quick UI building without designing from scratch.
<hr/>

### Material UI

A React component library based on Google’s Material Design guidelines.<br/>

Comes with polished components like dialogs, cards, buttons, and forms.<br/>

Highly customizable and easy to integrate with React projects.
<hr/>

### Ant Design

A professional React UI library commonly used in enterprise dashboards and admin panels.<br/>

Provides clean, modern, and data-heavy components like tables, charts, and forms.<br/>

Great for complex business apps.
<hr/>

### Chakra UI

A simple, accessible, modular React UI library.<br/>

Comes with built-in dark mode support and easy customization.<br/>

Very developer-friendly for building modern interfaces quickly.
<hr/>

### Styled components

A CSS-in-JS library that lets you write CSS directly inside your JavaScript files.<br/>

Styles are scoped to components, so no class name clashes.<br/>

Allows dynamic styling using props and supports theme customization.<br/>

Great for React apps that want clean, modular, component-based styling.
<hr/>

## TailwindCSS

A utility-first CSS framework that provides small classes like p-4, m-2, bg-blue-500, etc.<br/>

Lets you style everything directly in the JSX without writing separate CSS files.<br/>

Highly customizable through tailwind.config.js.<br/>

Helps build designs faster and more consistently across components.<br/>

No predefined UI components—complete flexibility.<br/>

Reduces unused CSS through PurgeCSS, making the bundle smaller.<br/>

Ideal for modern React and frontend projects that prefer custom designs over frameworks.
